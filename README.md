# 🚀 Sorting Algorithm Visualizer (Dockerized)

An interactive **Sorting Algorithm Visualizer** built using HTML, CSS, and JavaScript, containerized using Docker for easy deployment and portability.

---

## 🌟 Features

* Visualizes sorting algorithms step-by-step
* Clean and responsive UI
* Lightweight and fast
* Fully containerized using Docker

---

## 🐳 Docker Setup

### 🔹 Pull the Image

```bash
docker pull outroverse/sorting-visualizer
```

### 🔹 Run the Container

```bash
docker run -p 8080:80 outroverse/sorting-visualizer
```

### 🔹 Open in Browser

```
http://localhost:8080
```
